Key Features

🤖 ML-Powered Anomaly Detection

3-method ensemble scoring: Isolation Forest trained on baselines, Z-score statistical analysis, and Exponential Moving Average deviation tracking

📊 Real-Time Dashboard

Vanilla JavaScript SPA with WebSocket-powered updates, glassmorphism UI, system health visualization and resource monitoring

⚡ Automated Response

IP blocking via iptables with automatic reversal, process termination with safety guardrails, and configurable action limits per window

📋 Policy Engine

YAML-based alert rules with severity levels (critical, high, medium, low), cooldown periods, and hot-reload without restart

📡 Webhook Notifications

HMAC-signed alerts to Slack, Discord, or any HTTP endpoint with retry queue, delivery tracking, and configurable templates

🐳 Zero Host Agents

Everything runs in Docker; reads host data via bind-mounted /proc — no installation on host required, minimal footprint

🔗 Full REST API

9 REST endpoints plus WebSocket for programmatic access to metrics, alerts, actions, processes, ports, and logs

🔒 Docker Native Security

10 containerized services on private network (sentinel-net); single port exposure (8080), read-only mounts, non-root containers
